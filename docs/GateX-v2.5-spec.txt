#####################################################################
4. DETERMINISTIC GATE PACK
#####################################################################

4.1 Schema Gate
- Reject if mandatory fields missing
- Reject if numeric types invalid

4.2 Cooldown Gate
IF recent_failures >= threshold
  -> FAIL (DENY_COOLDOWN)

4.3 Budget Gate
IF today_spend + this_cost > daily_budget
  -> FAIL (unless urgency high in lenient mode)

4.4 Risk Gate
IF risk.score >= risk_deny
  -> FAIL
ELSE IF risk.score >= risk_warn
  -> WARN

4.5 Slippage Gate
IF slippage_bps >= slippage_deny_bps
  -> FAIL

4.6 Fee Ratio Gate
fee_ratio = fee_usd / expected_value_usd
IF fee_ratio >= fee_ratio_deny
  -> FAIL

4.7 Failure Density Gate
IF rolling_failure_rate >= failure_density_deny
  -> FAIL

4.8 Overtrade Gate
IF recent_actions_5m > overtrade_threshold
  -> WARN or FAIL

4.9 Max Size Gate
max_size_ratio adjusted downward based on:
- risk.score
- volatility
- rolling_failure_rate

4.10 Execute Gate (Final AND)
IF any FAIL
  -> DENY
ELSE ALLOW (possibly with clamp)

#####################################################################
6. ADAPTIVE TUNING ENGINE
#####################################################################

Tuning Frequency:
Every K requests (default: 25)

Adaptive Targets:
- rolling_failure_rate target: <= 8%
- rolling_deny_rate target: <= 80%

6.1 Conservative Shift Trigger
IF rolling_failure_rate > 12%
  risk_deny -= 0.02
  slippage_deny_bps -= 10
  max_size_base *= 0.9
  cooldown_base_sec *= 1.2

6.2 Relaxation Trigger
IF rolling_deny_rate > 85%
AND rolling_failure_rate < 6%
  risk_deny += 0.01
  max_size_base *= 1.1
  cooldown_base_sec *= 0.9

6.3 Volatility Amplification
IF avg_volatility > 0.7
  slippage_deny_bps *= 0.85
  max_size_base *= 0.8

6.4 Capital Stress Detection
IF position_size_usd / balance_usd > 0.6
  max_size_ratio *= 0.7

6.5 Insurance Mode Escalation
IF insurance_mode == true
  multiply all clamps by safety_factor (e.g., 0.8)
